## **소개 🐘**
코끼리끼리는 함께 목표를 달성하는 로드맵 중심 커뮤니티 플랫폼입니다. 지식 공유와 단체 활동을 통해, 여러분의 목표를 실현할 수 있도록 도와드립니다. 로드맵을 통해 지식을 공유하고, 골룸에서 같은 목표를 가진 사람들과 연결되어 함께 성취해 나가세요!
